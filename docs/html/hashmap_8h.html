Initialize a new hashmap with the options defined here, check the struct hashmap_container for details on what each one does.

Parameters
init_lenHow many nodes worth of space to pre-allocate
nonceAppend 64 bytes of this value to keys before hashing, if null nothing is appended.
load_limMaximum load factor (used slots/currently allocated) before triggering expansion.
expand_factorWhen expansion is triggered expand by this factor, recommended is 2.
Returns
